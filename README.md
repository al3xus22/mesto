# Проект: Место

Сайт Место.
Страница с возможностью редактирования профиля, добавления своих фотографий различных мест, а также лайкать их.

В проекте применены технологии:
* макет - Figma
* HTML
* CSS
* JavaScript
* ES6 Modules
* ООП
* Webpack

* [Ссылка на проект](https://al3xus22.github.io/mesto)
